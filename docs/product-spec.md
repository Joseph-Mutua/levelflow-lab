# Product Spec

## Positioning

FlowLab gives IT operators a safe staging layer for automation workflows in a remote monitoring and management platform.

## Primary Users

- IT admins validating risky automations before rollout
- MSP operators managing multiple customer organizations
- Technical support teams debugging failed remediations

## MVP Screens

1. Fleet Overview Dashboard
2. Automation Builder
3. Blast Radius Preview
4. Step-by-Step Simulation Console
5. Run Replay / Historical Debugger
6. AI Assistant Embedded Panel

## Core Objects

- Device
- Workflow
- Trigger
- Action node
- Condition
- Retry policy
- Simulation result
- Historical run
- AI insight

## Success Criteria

- Operators can understand which devices a workflow would affect before running it.
- Operators can see exactly why devices matched or were excluded.
- Operators can inspect step-level inputs, outputs, variables, and decisions.
- Operators can compare failed historical runs with prior successful runs.
- AI assistance is grounded in workflow context rather than acting as a generic chatbot.
